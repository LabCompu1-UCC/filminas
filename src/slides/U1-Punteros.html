El Puntero o apuntador

Los punteros son variables cuyos valores son direcciones de memoria.

Punteros

  • Una variable contiene un valor especifico
  • El puntero contiene la direccion de una variable que contiene un valor especifico.
  • Las variables hacen referencia directa a un valor.
  • Los punteros hacen referencia indirecta a un valor.

Punteros

Sintaxis


        int *miPuntero;

Punteros

Operador dirección

El operador dirección (&) es un operador uniario que obtiene la dirección de memoria de su operando. 

        int i = 5;
        int *iPtr = & ;

Punteros

ejemplo de uso:


#include <iostream>

int main(){
    int *miPuntero;
    int num = 5;

    cout << num << endl;
    cout << &num << endl;

    miPuntero = &num ;

    cout << miPuntero << endl;
    cout << *miPuntero << endl;

    return 0;
}
